# Marketing & SEO Audit

Analyze the landing page for search engine optimization, marketing effectiveness, and conversion potential.

## SEO On-Page Checklist

### 1. Technical SEO
- [ ] **Title Tag** (50-60 chars)
  - Current: "Create Next App"
  - Recommended: "Dr. João da Silva - Agende Consulta Online | Clínico Geral"
  
- [ ] **Meta Description** (150-160 chars)
  - Current: "Generated by create next app"
  - Recommended: "Consulta com clínico geral experiente. Atendimento online e presencial. Agende sua consulta agora mesmo."

- [ ] **URL Structure**
  - Clean, semantic URLs
  - Avoid query parameters if possible

- [ ] **Open Graph Tags** (for social sharing)
  - `og:title`: "Dr. João da Silva"
  - `og:description`: Consultation description
  - `og:image`: Doctor photo/branding (when available)
  - `og:type`: "website"

- [ ] **Twitter Card** (for Twitter/X sharing)
  - `twitter:title`: Page title
  - `twitter:description`: Summary
  - `twitter:image`: Featured image

- [ ] **Structured Data (Schema.org)**
  - Add `LocalBusiness` schema for medical professional
  - Include `medicalBusiness`, `areaServed`, `potentialAction`

### 2. Content & Keywords
- [ ] **Primary Keyword**: "Clínico geral + city name" (e.g., "clínico geral São Paulo")
- [ ] **Long-tail Keywords**: "Consulta online clínico geral", "Agendar consulta médica"
- [ ] **Keyword Placement**:
  - H1: ✅ "Dr. João da Silva"
  - Subheadings: ✅ "Clínico Geral • CRM"
  - Body: ✅ "Consulta Online & Presencial", "atendimento"
  - Meta: ❌ Needs update

- [ ] **Content Length**: ~150-200 words (current is good for landing page)
- [ ] **Keyword Density**: 1-3% (avoid keyword stuffing)

### 3. User Engagement Signals
- [ ] **Headings Hierarchy**: H1 → H2 (proper structure)
- [ ] **Readability**:
  - Average line length: ~60-80 chars ✅
  - Paragraph length: Short, scannable ✅
  - Use of lists: ✅ (bullet points with checkmarks)

- [ ] **Call-to-Action (CTA)**:
  - Primary CTA: "Solicitar agendamento" ✅
  - CTA visibility: Prominent, above-fold ✅
  - CTA clarity: Clear action ✅

- [ ] **Media**:
  - Add high-quality doctor photo (builds trust)
  - Consider adding before/after examples (if applicable)

### 4. Mobile Optimization
- [ ] **Mobile-First Design**: ✅ Responsive layout
- [ ] **Page Speed**: Test with Google PageSpeed Insights
  - Aim for 90+ score
  - Optimize images (WebP format)
  - Minimize CSS/JS

- [ ] **Touch Targets**: 44×44px minimum ✅
- [ ] **Viewport Meta Tag**: ✅ Present in layout.tsx

### 5. Link Strategy
- [ ] **Internal Links**: Link to other pages (if multi-page site)
- [ ] **External Links**: Link to relevant medical resources (builds authority)
- [ ] **Backlinks**: Encourage local business listings, directories
